## 下载

### Docker

```
docker pull apachecn0/flygon-sec-zh
docker run -tid -p <port>:80 apachecn0/flygon-sec-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-sec-zh
flygon-sec-zh <port>
# 访问 http://localhost:{port} 查看文档
```
